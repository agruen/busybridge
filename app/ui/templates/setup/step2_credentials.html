Google Cloud Credentials

Enter your OAuth 2.0 credentials from Google Cloud Console.

  1. Go to Google Cloud Console
  2. Create a new project or select an existing one
  3. Go to "APIs & Services" > "Library"
  4. Search for and enable "Google Calendar API"
  5. Go to "APIs & Services" > "Credentials"
  6. Click "Create Credentials" > "OAuth client ID"
  7. Select "Web application" as application type
  8. Add authorized redirect URIs:
    • https://your-domain/auth/callback
    • https://your-domain/auth/connect-client/callback
    • https://your-domain/setup/step/3/callback
  9. Copy the Client ID and Client Secret
