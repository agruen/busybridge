Welcome to Calendar Sync

Let's get your calendar synchronization service set up.

What This App Does

Calendar Sync keeps your main calendar synchronized with multiple client organization calendars:

  • Events from client calendars appear on your main calendar with full details
  • Your main calendar events appear as "Busy" blocks on client calendars
  • Changes sync bidirectionally and automatically

Before You Begin

Make sure you have:

  • A Google Cloud project with Calendar API enabled
  • OAuth 2.0 credentials (Client ID and Secret)
  • Admin access to your Google Workspace

Open Google Cloud Console
